Crear un Cuestionario

Esta función sirve para generar un cuestionario introduciendo archivos de audio y si es necesario, también se puede introducir archivos multimedia.

• Se deben introducir los nombres de los lugares además de ponerle un nombre al Cuestionario, esto se hace para facilitar la tarea de rellenar los cuestionarios a los encuestados.

• El sistema de creación de los cuestionarios es muy sencillo, se selecciona el número de lugares y el número de sistemas de grabación o grabaciones distintas por lugar.

Rellenar un Cuestionario

Después de crear los cuestionarios se pueden estos rellenar por parte de los partipantes. Los cuestionarios que muestra esta aplicación son "Cuestionarios de Paisajes Sonoros".

Un cuestionario sonoro es una lista de preguntas con el fin de evaluar el entorno acústico de un lugar tal y como es percibido, experimentado o entendido por las personas.

Todos los cuestionarios creados a través de esta aplicación siguen la siguiente estructura:

• Preguntas Generales al Participante

     -- En esta parte se le pregunta al usuario cosas sobre si mismo tales como la edad, el género o si tiene problemas auditivos.

Las personas con problemas auditivos no pueden rellenar este cuestionario al distorsionar los resultados finales.

• Preguntas de Introducción al Escenario

     -- En esta parte se le pregunta al usuario sobre la ubicación donde se han realizado las grabaciones tales como si conoce el lugar o su frecuencia de paso. El usuario tendrá que responder esta parte tantas veces como lugares a examinar tenga.

• Preguntas Individualizadas

     -- En esta parte se le pregunta al usuario características que tendrán las grabaciones de cada lugar.

Por cada lugar al menos debe haber una grabación. Estas preguntas aparecerán cada vez que el Participante escuche una grabación.

• Preguntas sobe la Idoneidad del Entorno Sonoro Circundante

     -- En esta parte se le pide al usuario que después de haber escuchado todas las grabaciones ordene de más a menos cada toma.

Algunas características que se preguntan son "De más a menos ruidosa" o "De más desagradable a menos desagradable".

Luego de acabar con la parte de la última parte se da por terminado el cuestionario.
